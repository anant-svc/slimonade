# slimonade 
### It is combination of [Slim Framework|https://www.slimframework.com/] and [LemonadeJS|https://lemonadejs.net/]
-
Runs on PHP 8.1 

Features available:
* Fastest PSR 7 routing
* PSR 3 Logger
* Theme based fallback
* Lazy PDO
* Modular
* Session manager
* In-built Error handler
* No complex code
